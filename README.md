Simon Game (C++ / Qt)

This project is a desktop implementation of the classic Simon memory game built using C++ and the Qt framework. 
It was developed as part of CS 3505 – Software Practice II at the University of Utah.

The application challenges the player to repeat progressively longer sequences of button inputs generated by the computer. 
The project focuses on event-driven programming, GUI development, and separation of application logic from user interface components.

Features -----
Graphical user interface built with Qt Widgets
Event-driven architecture using Qt signals and slots
Visual feedback through button highlighting animations
Sound effects for correct input, incorrect input, and computer sequence playback
Progress tracking using a dynamic progress bar
Game state management including start, gameplay, and game over states

Architecure -----

The project follows an MVC-inspired design pattern:

Model (SimonModel): Responsible for game logic, sequence generation, and state management
View (MainWindow): Handles rendering of the user interface and visual feedback
Controller: Implemented through Qt’s signals and slots system, connecting user actions to model updates

This separation allows for a clear distinction between presentation and logic, improving maintainability and readability.

Technologies Used -----
C++
Qt Framework (Widgets)
QSoundEffect for audio feedback
QTimer for UI timing and animations
Signal/slot-based event system
